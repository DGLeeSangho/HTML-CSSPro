1. CSS
	ㄱ. [C]ascading [S]tyle [S]heets 약어
	ㄴ. html 요소가 표시되는 방법을 설명
	ㄷ. html 문서의 스타일 지정하는데 사용되는 언어
	ㄹ. 많은 작업( 유지, 보수, 지정 ) 절약
	ㅁ. 여러 웹 페이지의 스타일을 일괄적으로 관리할 수 있다.( 장점 )
	ㅂ. 외부 스타일 시트 ( .css ) 사용
	ㅅ. 다양한 장치(기기) 및 화면 크기에 대한 출력의 디자인, 레이아웃 및 변경 스타일 정의 처리
	ㅈ. html 요소가 화면, 종이 또는 기타 미디어에 표시되는 방식 지정.
	
2. CSS구문(형식)
	스타일 적용 대상						세미콜론(;)
		selector  		{ 속성:속성값;속성:속성값;속성:속성값;...}
		선택자
		
3. CSS 적용하는 방법
	ㄱ. 내부( Internal ) CSS 적용 방법 - 해당 문서(웹 페이지)만 스타일 적용
		head
			style
			/style
		/head
	ㄴ. 인라인( inline ) CSS 적용 방법 - 해당 요소에만 스타일 적용
		시작태그 안에 style 속성으로 스타일 적용
		p  style="스타일 적용"
	ㄷ. 외부 ( External ) CSS 적용 방법 - 여러 문서(웹 페이지)에게 스타일 적용
		외부 스타일 적용할 파일
		style.css
		
4. CSS 선택자( selector )
	ㄱ. 태그명
	ㄴ. #아이디명
	ㄷ. .클래스명
	ㄹ. 그룹화
