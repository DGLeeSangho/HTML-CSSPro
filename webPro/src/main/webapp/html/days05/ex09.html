html 반응형 웹

	1. 모든 장치에서 웹 페이지가 보기 좋게 만드는 것.
	2. 다양한 화면 크기와 뷰포트에 맞게 자동으로 조정되게 하는것
	3.  코딩추가
	4. 반응형 웹 디자인
		1) 반응형 이미지 처리
		너비(width) 속성 : 퍼센트 %
		
		커피이미지 크기 : 너비 100px ,  높이200px;
		
		장치 크기 : 너비 400px
		커피이미지  width : 100%  해상도 깨진다.
							max-width: 100%  100px이상 너비가 확대 X
		
		2) 반응형 텍스트 크기 : vw   				[v]iewport [w]idth
			

			1vw == 뷰포트 1%
			휴대폰 장치 너비 / 뷰포트 50cm = 1vw == 0.5cm
			
		3) 미디어 쿼리 - 를 사용해서 텍스트와 이미지의 크기를 조정
			
			예) ex06_02.html

			http://192.168.0.14/webPro/html/days05/ex06_02.html
