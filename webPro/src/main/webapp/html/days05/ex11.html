	[ html 코딩 규칙( 가이드 ) ]
	1. 문서 유형 선언 : 대문자 권장  !DOCTYPE html
	2. 요소(태그)는 소문자 권장
	3. 모든 요소가 종료태그가 필요한 것은 아니다.
		br, hr, meta, img  X
	4. img src="" alt="" 등호 주위에 공백 사용 X
	5. p    /p   꼭 닫기태그 코딩 하자.
	6. 폴더명, 파일명은 *** 대소문자 구분을 한다 ***
		요청 URL -> 웹서버( 톰캣 ) : 대소문자 구분
	7. 파일 확장자 			:   .html    		.htm
		외부 css 파일			:   .css
		외부 javascript파일:  .js
