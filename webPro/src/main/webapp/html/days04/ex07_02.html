1. 모든 html 요소(태그)에는 요소 유형에 따라 기본 표시(display) 값이 있다.
2. 표시 값에는 2가지가 있다. ( 블럭모드 , 인라인 모드 )
- 블럭모드 : 요소 전체 너비를 차지 p, div, li, hn, dl, dd, dt, ol, ul, hr, blockquote 등등
- 인라인모드 : 새 줄에서 시작 X, 필요한 너비 만큼만 차지한다. span, img, a, map, b, button, i, q, sub, sup, input 등등

div 요소(태그)

1. 웹 페이지의 배치 ( layout )
2. 블럭모드
3. 일반적으로 컨테이너의 종류 중에 하나
